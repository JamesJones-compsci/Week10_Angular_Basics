Date Pipe Examples

Today : {{ today }}

  • Short Date: {{ today | date:'shortDate' }}
  • Medium Date: {{ today | date:'mediumDate' }}
  • Long Date: {{ today | date:'longDate' }}
  • Full Date: {{ today | date:'fullDate' }}
  • Long : {{ today | date:'long' }}
  • Medium : {{ today | date:'medium' }}
  • Short : {{ today | date:'short' }}
  • Custom (dd-MM-yyyy): {{ today | date:'dd-MM-yyyy' }}

Currency Pipe Examples

  • Default : {{ amount }}
  • Currency : {{ amount | currency }}
  • USD: {{ amount | currency:'USD' : 'code' }}
  • EUR: {{ amount | currency:'EUR' }}
  • GBP: {{ amount | currency:'GBP' }}
  • AER: {{ amount | currency:'AER' }}
  • JPY: {{ amount | currency:'JPY' }}
  • INR: {{ amount | currency:'INR' }}
  • CAD (narrow symbol): {{ amount | currency:'CAD' : 'symbol-narrow' }}

Number Pipe Examples

  • Default : {{ amount }}
  • 2 decimal places : {{ 12.93836 | number:'1.2-2' }}
  • minimum 3 integer digits: {{ 12.93836 | number:'3.0-0' }}
  • Percentage: {{ 15.89 | percent }}
  • Percentage: {{ 0.1589 | percent }}
